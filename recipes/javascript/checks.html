Checks

Run format, lint, and test checks using Node.js

Recipes / JavaScript / Checks

This works whether you will have the app run with Node in the browser.

Some commands are for making changes and some are for running a check only (useful in a CI flow when you want logged output but don‚Äôt actually want to make the commit as that makes more effort to set up and might not be suitable to apply).

Samples

Based on package.json from peaceiris/actions-gh-pages repo.
