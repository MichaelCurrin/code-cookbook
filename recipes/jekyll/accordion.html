Accordion

Recipes / Jekyll / Accordion

Details HTML tag

Use the details and summary tags - these work on Github markdown, Jekyll, Docsify and others. No CSS or JS need.

Simple use

Code

<details>
<summary><b>My title</b></summary>
My expanded details
</details>

Result

My title My expanded details

Nesting

You can nest accordion items.

Code

<details>
<summary><b>Outer title</b></summary>
Outer details
<details>
    <summary><b>Inner title</b></summary>
    Inner details
    </details>
</details>

Result

Outer title Outer details
Inner title Inner details

Just be sure to not use indenting for the inner item, or just make sure there are no empty lines as above. Otherwise your indented HTML becomes markdown code blocks.

If you need to style the inner and outer items differently, you can add divs or classes or a CSS selector which only applies to details inside details.

Jekyll templating use

Create an includes file and use it on a page.

  • _includes/accordion.html - or whatever name you like. 
      <details>
     <summary>
         <b>{{ include.title }}</b>
     </summary>
     {{ include.description }}
  </details>
  • index.md 
      ---
  layout: null
  ---

  # Accordion test

  {% include accordion.html
      title="This is my title"
      description="This is my description"
  %}

Plugin

See also Creating an Accordion Plugin for Jekyll - No TV and No Beer 6 - this is a heavier approach which involves writing two Ruby plugins.

Custom CSS approach

This is similar to using the Details tag but uses HTML tags with custom CSS - which is a lot heavier but allows more customization. This approach can be used with an includes files too.

From Implementing A Pure CSS Collapsible

  • accordion.html 
      <div class="wrap-collapsible">
      <input id="collapsible" class="toggle" type="checkbox">
       
      <label for="collapsible" class="lbl-toggle">
          {{ include.title }}
      </label>

      <div class="collapsible-content">
          <div class="content-inner">
              <p>
                  {{ include.description }}
              </p>
          </div>
      </div>
  </div>
  • index.md 
      ---
  layout: null
  ---

  # Accordion test

  {% include accordion.html
      title="This is my title"
      description="This is my description"
  %}
